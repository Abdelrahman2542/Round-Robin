# CPU Scheduler Simulator - Round Robin ⏱️

This project is an Operating Systems CPU Scheduling Simulator, specifically implementing the **Round Robin (RR)** algorithm in Python. 

The project includes two parts:
1. **Core Template Simulator**: A CLI-based template simulator used for automatic testing and standard execution.
2. **GUI Bonus**: A graphical user interface (GUI) built with `tkinter` to visually simulate the processes and generate Gantt Charts.

## 📁 Repository Structure

- `simulator/`: Contains the core logic of the scheduling algorithms. The Round Robin logic is implemented inside `simulator/schedulers/rr.py`.
- `examples/`: Contains text files for testing the simulator (e.g., `test1.in`).
- `schedules/`: The output text files generated by the simulator will be saved here (e.g., `RR.txt`).
- `GUI/`: Contains the bonus graphical interface script `round_robin.py`.
- `requirements.txt`: Lists any Python package dependencies required to run the CLI simulator.

---

## 🚀 How to Run the Project

### 1. Running the CLI Simulator (Official Template)
This is the standard execution method that reads an input file and writes the context switch logs and average waiting time to an output file.

**Prerequisites:**
You may need to install the requirements first:
```bash
pip install -r requirements.txt
```

**Execution:**
Run the simulator by passing the path to a test file:
```bash
python -m simulator examples/test1.in
```
*Note: The input file should have the format `<process id> <arriving time> <burst time>` for each line.*

**Output:**
The result will be generated inside the `schedules/` folder as `RR.txt`.

### 2. Running the GUI Application (Bonus)
If you want to see a beautiful visual representation of the Round Robin algorithm with a Gantt chart, you can run the GUI version.

**Execution:**
```bash
python GUI/round_robin.py
```
- A window will pop up.
- You can manually add processes or click **"Load Example"** to fill in sample data.
- Click **"Run Scheduler"** to see the Gantt chart and statistics (Average Waiting Time, Turnaround Time, etc.).
